ClawPanel

OpenClaw AI Agent 框架的可视化管理面板，基于 Tauri v2 构建的跨平台桌面应用。

下载最新版 源代码

功能特性

仪表盘

系统运行概览，服务状态实时监控，一目了然掌握 OpenClaw 运行情况。

服务管理

OpenClaw 服务启停控制、版本检测与一键升级、配置备份与恢复。

模型配置

多服务商管理、模型增删改查、主模型选择、批量测试、延迟检测，支持自动保存与撤销。

网关配置

Gateway 端口、运行模式、认证方式可视化配置，即改即生效。

日志查看

多日志源实时查看与关键词搜索，快速定位运行异常。

记忆管理

OpenClaw 记忆文件的查看、编辑、分类管理与 ZIP 导出。

扩展工具

cftunnel 内网穿透管理、ClawApp 连接状态监控等扩展功能。

主题切换

支持暗色与亮色主题切换，适应不同使用环境偏好。

技术架构

层级技术说明
前端Vanilla JS + Vite零框架依赖，轻量快速
后端Rust + Tauri v2原生性能，跨平台打包
通信Tauri IPC + Shell Plugin前后端桥接，本地命令执行
样式纯 CSS (CSS Variables)暗色主题，玻璃拟态风格

快速开始

前置条件

需要安装 Node.js >= 18、Rust stable 以及 Tauri v2 系统依赖。

# 克隆仓库
git clone https://github.com/qingchencloud/clawpanel.git
cd clawpanel
npm install

# 启动开发模式（完整 Tauri 桌面应用）
./scripts/dev.sh

# 仅启动前端（浏览器调试，使用 mock 数据）
./scripts/dev.sh web

# 构建发布版本
./scripts/build.sh release
